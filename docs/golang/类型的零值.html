数字、布尔、字符串

对于这三种类型，他们的零值分别是 0、false、""（空字符串）

package main

import "fmt"

func main() {
    var num int
    var str string
    var boolean bool
    fmt.Println(num, str, boolean)
}

输出：
0  false

结构体

结构体的零值就是结构体中每个字段的零值

package main

import "fmt"

type Out struct {
    inner   Inner
    num     int
    boolean bool
}

type Inner struct {
    num     int
    boolean bool
}

func main() {
    var out Out
    fmt.Println(out)
}

输出：
{{0 false} 0 false}

首先输出 inner 的零值，也就是内部两个字段的零值，接着输出 out 本身其它两个字段的零值

接口和引用

接口和引用类型（slice、指针、map、通道、函数）的零值是 nil

package main

import (
    "fmt"
)

func main() {
    var aSlice []int
    var aPointer *int
    var aMap map[int]int
    var aChannel chan int
    var aFunc func(int) int
    fmt.Println(aSlice == nil, aPointer == nil, aMap == nil, aChannel == nil, aFunc == nil)
}

输出：
true true true true true
